Indicator Reference

Token / Alias Type Usage Example
Price, LTP, CMP Numeric Price > [Daily].High
RSI Numeric RSI < 30 OR [1h].RSI > 60
Trend, ST, SuperTrend Sentiment Trend == 'Bullish'
ST_V, STV, ST_Value Numeric LTP > ST_V + 0.5%
Pattern Sentiment Pattern == 'Bullish'
PatternScore, PScan Numeric PatternScore >= 3
EMA_F, EMA_S, EMA_C Mixed EMA_C == 'Bullish'
Vol, VOL_R, VolumeRatio Numeric VOL_R > 2.5

Operators

  • AND / OR / NOT - Logical gates.
  • == / != - Equality (use single quotes for strings).
  • > / >= / < / <= - Numeric range.

Percentage Engine

Apply offsets directly to any price-based token. This is used for "buffer" zones or trailing targets.

LTP > ST_V + 0.5%
High - 1.25%
LTP < [1h].EMA_S - 2%
Special Logic: Target & Stop-Loss

In Target/SL fields, if you leave them empty, the system defaults to 1.5% profit and 1.5% loss based on LTP. You can use tokens for dynamic exits:

Target: [Daily].High
Stop Loss: ST_V - 0.25%
Shorthand Power Tokens
@High → Previous Candle High
@Low → Previous Candle Low

Timeframe Wrappers

The power of Confluence Lab lies in checking multiple timeframes simultaneously. Wrap any token in brackets [TF] to jump contexts.

[5m] [15m] [1h] [60m] [Daily] [Weekly] [Current]
